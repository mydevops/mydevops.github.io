[Effective Python 01] 关于 Python 版本的选择

Posted on 2017-03-01   |   In python

Python主要分为两大派系Python 2系列和Python 3系列，虽然Python 2以及Python 3 Python社区的积极维护中，但是Python 2的功能开发已经被冻结，只会进行bug修复、安全增强以及移植等工作，以便于开发者能够顺利的从Python 2迁移到Python 3。2to3与six等工具都可以使Python 2的代码转换为Python 3的代码上，但是并不推荐使用。

Python 3经常会添加新功能并提供改进，而这些功能与改进并不会兼容Python 2。并且绝大部分的开源代码块都兼容Python 3，所以说希望大家能够逐渐的转向Python 3。

关于Python的版本选择上，应该很多人都会有疑惑，很多操作系统中都默认安装了Python，运行Python的默认命令是python。可以使用--version或者-V参数来确定Python的版本。

Python的解释器有很多种，CPython、Jython以及PyPy等，我们平常所说的Python都是默认指的是CPython。

Python 2 和 Python 3 的简要区别

简单列举几个Python 2和Python 3的几点区别，来试试手Python 3的魅力。

print函数

Python 2中的print语句被Python 3中的print()函数取代，所以说在Python 3中必须使用括号将要输出的内容括起来。
